* ```python
  x = ['a', 'b', 'c']
  x[-1:] = [3]
  print(x)
  ```

  这个考的python，答案应该是['a', 'b', 3]。

* 在长度为6的**循环队列**中，若`front`=5，`rear`=2，添加两个元素，删除一个元素，则`front`和`rear`应该是0和4。

  队列queue都是先进先出的，添加元素是在队尾，删除元素是在队末，添加两个元素`rear+=2`，删除一个元素`front+=1`再取模所以是0。

  再复习一下循环队列判断空和满的条件。空是`front=rear`，满是`front=(rear+1)%maxLen`

* 关于swap函数的问题，涉及到指针和引用（害 这我总是搞不清，下次一定要特别整理一篇指针和引用！！！

  ```C++
  void swap1(int &x, int &y){
  	int tmp;
  	tmp = x;
  	x = y;
  	y = tmp;
  }
  
  void swap2(int x, int y) {
      int temp = y;
      y = x;
      x = temp;
  }
  
  void swap3(int *x, int *y){
      int *tmp;
      *tmp = *x;
  	*x = *y;
  	*y = *tmp;
  }
  
  void swap4(int *x, int *y){
      int tmp;
      tmp = *x;
  	*x = *y;
  	*y = tmp;
  }
  
  ```

  第一个就是最常用的swap函数，是可以实现交换的。

  第二个是最常见的错误了，只改变了形参，实参是没有改变的。

  第三个会发生Segmentation Fault，这是因为定义的整型指针变量tmp没有赋初始值。任何指针变量刚被创建时不会自动成为NULL指针，它的缺省值是随机的，就会随便指向一个地址。所以**指针变量在定义的同时就应该被初始化**：1.设空值，NULL；2.指向合法地址。这种称为野指针，是没有访问权限的指针，所以不能把x的值赋给它。

  第四个传入的参数是指针，也就是x和y内存的地址。这就可以直接修改指针指向的内存，从而实现交换。

* ++x和x++的区别

  ```C++
  int a=10, b=20, c=30, res;
  ++a;
  a++;
  res = ++a + (++b) + (c++) + a++;
  ```

  返回值不一样，++x返回的是x+1后的值，x++返回的是+1之前的值。所以res是77。

* ```python
  a = [1, 3, 5, 7]
  a.sort(key=lambda a:len(str(a)), reverse=True)
  ```

  问的是a。刚才跑了一下，发现a是不变的。

  这题让我有点懵。首先是list的sort函数，有两个参数，一个是key，接收一个函数作为参数；另一个是reverse，这个很好理解，True是从大到小，False是从小到大，默认是False。

  key这个参数的例子如下，takeSecond这个函数返回tuple的第二个值，key取takeSecond就是让tuple按第二个值从小到大排序。

  ```python
  def takeSecond(elem):
      return elem[1]
  
  random = [(2, 2), (3, 4), (4, 1), (1, 3)]
  random.sort(key=takeSecond)
  # OUTPUT: [(4, 1), (2, 2), (1, 3), (3, 4)]
  ```

  lambda函数我之前看到过但不是很了解，只知道是定义匿名函数的方法。现在查了资料知道了冒号前是函数的返回值，后面是函数体。在这里list的四个元素都是个位数，所以返回的都是1。实践表明key相同的时候，无论reverse是True还是False，都不改变元素的顺序（是不是可以说内部实现是一种稳定排序呢）。

  顺便提一句，`str(a)`居然就是`[1, 3, 5, 7]`（带空格的，哪怕代码里给a赋值的时候不带空格，转成str也会自动给你加上），所以`len(str(a))`是12。当然，这些都和这题无关了。

* $X_i$服从$\lambda=2$的指数分布，$X = \sum_{i=1}^{32}X_i$，$a=P\{X\leq 16\}$，$b=P\{X\geq 12\}$，问a和b的大小关系

* 还有一道给了三个$5*5$的像素表格，一个是原始数据，两个是用不同滤波方法得到的结果，问各是什么滤波方法。选项里有提到高斯滤波、均值滤波、双边滤波和中值滤波。

  均值滤波和中值滤波望文生义嘛。

  双边滤波是高斯滤波+欧氏距离的结合。

  记某位置像素点的大小是$f(\xi)$，$c(\xi, x)=e^{\frac{1}{2}(\frac{d(\xi, x)}{\sigma_d})^2}$是高斯分布的值，$d(\xi, x)=\parallel \xi -x \parallel$是欧氏距离。则高斯滤波的公式是：
  
  $$
  h(x) = k_d^{-1}(x) \int_{-\infty}^{\infty} \int_{-\infty}^{\infty} f(\xi) c(\xi, x) d\xi \\
  k_d(x) = \int_{-\infty}^{\infty} \int_{-\infty}^{\infty} c(\xi, x) d\xi
  $$
  
  其中，$k_d(x)$是归一化参数。
  
  而双边滤波的公式是这样的：
  
  $$
  h(x) = k^{-1}(x) \int_{-\infty}^{\infty} \int_{-\infty}^{\infty} f(\xi) c(\xi, x) s(f(\xi), f(x)) d\xi \\
  k(x) = \int_{-\infty}^{\infty} \int_{-\infty}^{\infty} c(\xi, x) s(f(\xi), f(x)) d\xi
  $$
  
	$s(f(\xi), f(x)) = e^{\frac{1}{2}(\frac{\delta(f(\xi), f(x))}{\sigma_r})^2}$是用像素值计算的高斯分布，$\delta(\phi, f)=\parallel \phi -f \parallel$是用像素值计算的欧氏距离，$k(x)$仍然是归一化参数。
